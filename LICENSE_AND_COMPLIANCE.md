
# LICENSE_AND_COMPLIANCE.md

## Purpose

This project uses open model checkpoints and open datasets. Codex must not assume blanket redistribution rights.

## Rules

1. Do not bundle pretrained weights into the repo.
2. Do not bundle the dataset itself into the repo.
3. Record exact model and dataset identifiers in run manifests.
4. Keep a short provenance note in the paper appendix.
5. Prefer open models and datasets with clearly documented public access.

## Expected assets to reference

- Pythia checkpoints [R6]
- FineWeb / FineWeb-Edu [R7][R8]

## If a fallback model or dataset is used
Document:
- source URL,
- license or terms note,
- reason for the substitution,
- and whether it changes the interpretation of results.
